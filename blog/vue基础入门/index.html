Vue基础入门

2025年5月26日 · 阅读需 11 分钟

Back-end engineers

1. 使用 data 提供数据​

  • data 属性是一个函数
  • 函数需要返回一个对象
  • 在模板中访问顶级属性: <div>{{prop}}</div>
  • 嵌套属性: <div>{{prop.username}}</div>
const app = Vue.createApp({
  data() {
    return {
      name: "张山",
    };
  },
});
app.mount("#app");
<div id="app">
  <h1>你好,{{name}}!</h1>
</div>

2. 在 html 属性中绑定 data 数据: v-bind

  • <a v-bind:href="link">链接</a>
  • <a :href="link">链接</a>
const app = Vue.createApp({
  data() {
    return {
      link: "https://gugulh.github.io/",
    };
  },
});
app.mount("#app");
<div id="app">
  <a v-bind:href="link">链接</a>
  <a :href="link">链接</a>
</div>

3. 展示列表内容: v-for​

  • 遍历数组类型数据
  • 可以访问索引:<li v-for="(todo, index) in todos">{{index + 1}} - {{todo}}</li>
  • 能够遍历对象数组
  • 可以在指令中使用 js 表达式
const app = Vue.createApp({
  data() {
    return {
      // todos: ["一起压马路", "一起吃饭", "一起溜pp"]
      todos: [
        {
          content: "一起吃饭",
          complete: true,
        },
        {
          content: "一起压马路",
          complete: true,
        },
        {
          content: "一起溜pp",
          complete: false,
        },
      ],
    };
  },
});
app.mount("#app");
<div id="app">
  <ul>
    <!-- <li v-for="(todo, index) in todos">{{index + 1}} - {{todo}}</li> -->
    <li v-for="(todo, index) in todos">
      <input type="checkbox" :checked="todo.complete" />{{todo.content}}
    </li>
  </ul>
</div>

4. 根据条件展示页面内容​

  • v-if, v-else-if, 和 v-else
  • 与 js 中 if, else...if, else 一样
  • 这三个标签必须相邻使用
  • v-show 会设置 display 属性为 none 隐藏元素
const app = Vue.createApp({
  data() {
    return {
      // books: ["JavaScript 基础语法详解", "Vue 入门实战", "React 入门到精通"],
      books: [],
      // books: ["JavaScript 基础语法详解"],
    };
  },
});
app.mount("#app");
<div id="app">
  <!-- <p v-if="books.length === 0">目前没有任何书籍</p>
    <h2 v-else-if="books.length === 1">{{ books[0] }}</h2>
    <ul v-else>
      <li v-for="book in books">{{ book }}</li>
    </ul> -->
  <p v-show="books.length === 0">目前没有任何书籍</p>
  <ul v-show="books.length > 0">
    <li v-for="book in books">{{ book }}</li>
  </ul>
</div>

5. 事件与响应入门​

  • v-on:事件名 或 @事件名
  • 值可以为 js 表达式或函数
  • 常见的注册事件
HTML 事件名Vue 事件名
onclickclick
onchangechange
onblurblur
onfocusfocus
onsubmitsubmit
onresetreset
const app = Vue.createApp({
  data() {
    return {
      showAnswer: false,
    };
  },
});
app.mount("#app");
<div id="app">
  <div class="question">
    <p>问：Vue 是一个什么样的框架？</p>
    <p v-show="showAnswer" class="answer">

    </p>
    <button @click="showAnswer = !showAnswer">

    </button>
  </div>
</div>

6. computed 计算属性​

  • 使用 computed 配置项定义计算属性
  • 每个计算属性都是一个函数
  • 函数可以使用 this 访问 data 或其他配置项中的值
  • 在模板中可以像 data 属性一样使用
const app = Vue.createApp({
  data() {
    return {
      showAnswer: false,
    };
  },
  computed: {
    label() {
      return this.showAnswer ? "隐藏答案" : "显示答案";
    },
  },
});
app.mount("#app");
<div id="app">
  <div class="question">
    <p>问：Vue 是一个什么样的框架？</p>
    <p v-show="showAnswer" class="answer">
      答：Vue 是一套用于构建用户界面的渐进式框架。
    </p>
    <button @click="showAnswer = !showAnswer">{{ label }}</button>
  </div>
</div>

优点: 避免在 html 中直接写计算逻辑,可以代码复用,减少冗余代码

7. methods 方法处理事件​

  • 使用 methods 配置项处理事件或复用逻辑
const app = Vue.createApp({
  data() {
    return {
      showAnswer: false,
    };
  },
  computed: {
    label() {
      return this.showAnswer ? "隐藏答案" : "显示答案";
    },
  },
  methods: {
    toggleAnswer() {
      this.showAnswer = !this.showAnswer;
    },
  },
});
app.mount("#app");
<div id="app">
  <div class="question">
    <p>问：Vue 是一个什么样的框架？</p>
    <p v-show="showAnswer" class="answer">
      答：Vue 是一套用于构建用户界面的渐进式框架。
    </p>
    <button @click="toggleAnswer">{{ label }}</button>
  </div>
</div>

8. watch 监听数据变化​

  • 使用 watch 定义监听器: watcher 可以是定时器,请求 API 等耗时操作
  • 监听器名字要和监听的属性名保持一致
  • 监听器接收属性旧值和新值作为参数
const app = Vue.createApp({
  data() {
    return {
      showAnswer: false,
      countDown: 5,
      timer: null,
    };
  },
  computed: {
    label() {
      return this.showAnswer ? "隐藏答案 " + this.countDown : "显示答案";
    },
  },
  methods: {
    toggleAnswer() {
      this.showAnswer = !this.showAnswer;
    },
  },
  watch: {
    showAnswer(newVal, oldVal) {
      if (newVal) {
        this.countDown = 5;
        if (this.timer) {
          clearInterval(this.timer);
          this.timer = null;
        }
        // 5秒后关闭答案
        this.timer = setInterval(() => {
          this.countDown -= 1;
          if (this.countDown === 0) {
            this.showAnswer = false;
            clearInterval(this.timer);
            this.timer = null;
          }
        }, 1000);
      }
    },
  },
});
app.mount("#app");
<div id="app">
  <div class="question">
    <p>问：Vue 是一个什么样的框架？</p>
    <p v-show="showAnswer" class="answer">
      答：Vue 是一套用于构建用户界面的渐进式框架。
    </p>
    <button @click="toggleAnswer">{{ label }}</button>
  </div>
</div>

9. computed, methods 和 watch 之间的区别​

  • computed 和 methods 之间的区别
    • 计算属性适合做一些简单的操作,结果会缓存
    • 计算属性可以直接在 html 中,像 data 中的属性一样使用
    • 方法适合做复杂的操作,通常作为事件监听或者公共业务逻辑
    • 方法可以当作普通 js 函数使用
  • computed 和 watch 之间的区别
computedwatch
简单的业务逻辑计算耗时操作和远程 API 加载
可以直接在 html 模板中使用不能在 html 模板中使用
响应 data 数据变化响应数据变化
有返回值/getter没有返回值
也可以使用 setters 修改 data 中的数据可以修改 data 中的数据
  • methods 和 watch 之间的区别
methodswatch
可以在 watch 中被调用不能调用
可以直接在 html 模板中使用不能在 html 模板中使用
响应 data 数据变化响应 data 数据变化
可以有返回值,也可以没有没有返回值
  • 应用场景
    • computed 适合根据 data 中的属性做一些简单的计算并返回结果,例如数组的排序筛选,只有 data 中的属性发生变化时,才会重新计算,其他情况会缓存结果
    • watch 适合在 js 中监听 data 的变化,并根据变化做一些耗时的操作
    • methods 适合更复杂的计算,通常作为事件处理函数使用或者是抽取公共的业务逻辑,当作普通的 js 函数

10. 表单输入数据处理​

  • 给 input 添加@input 事件
  • 在 input 事件监听中,修改 data 属性
const app = Vue.createApp({
  data() {
    return {
      username: "",
    };
  },
  methods: {
    handleInput(e) {
      this.username = e.target.value;
    },
  },
});
app.mount("#app");
<div id="app">
  <div>
    <div class="input-field">
      <label for="username">用户名：</label>
      <input id="username" type="text" @input="handleInput" />
    </div>
    <div class="result">
      <p>你输入的用户名是：{{ username }}</p>
    </div>
  </div>
</div>

11. 双向绑定和 v-model​

  • 双向绑定:输入框的值和 data 等属性值互相同步
  • v-model 简化了@input 和:value 数据的同步
const app = Vue.createApp({
  data() {
    return {
      username: "",
    };
  },
  methods: {
    handleInput(e) {
      this.username = e.target.value;
    },
    resetUsername() {
      this.username = "";
    },
  },
});
app.mount("#app");
<div id="app">
  <div>
    <div class="input-field">
      <label for="username">用户名：</label>
      <input
        id="username"
        type="text"
        @input="handleInput"
        :value="username"
        <!--
        v-model="username"
        --
      />
      />
      <button @click="resetUsername">重置</button>
    </div>
    <div class="result">
      <p>你输入的用户名是：{{ username }}</p>
    </div>
  </div>
</div>

14. 在 inline-style 内联样式中绑定 data 数据​

  • 直接使用 v-bind 绑定
  • 也可以绑定计算属性,方法等
const app = Vue.createApp({
  data() {
    return {
      fontSize: 16,
    };
  },
  computed: {
    pStyle() {
      return {
        color: "white",
        fontSize: this.fontSize + "px",
      };
    },
  },
});
app.mount("#app");
<div id="app">
  <div>
    <p :style="pStyle">这是一个段落</p>
    <button class="btn" @click="fontSize += 1">+</button>
    <button class="btn" @click="fontSize -= 1">-</button>
  </div>
</div>

15. 根据条件添加或删除 class 样式​

  • 直接使用 v-bind 绑定
  • 可以绑定单个 class
  • 可以用数组绑定多个 class
  • 可以用对象根据条件添加/移除 class
const app = Vue.createApp({
  data() {
    return {
      textClass: "text-blue",
      stateClass: "danger",
      isHidden: false,
    };
  },
});
app.mount("#app");
<div id="app">
  <p :class="[textClass, stateClass, { hide: isHidden }]">这是一个段落</p>
  <button @click="isHidden = !isHidden">切换状态</button>
</div>
