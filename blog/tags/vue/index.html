6. 事件传参与多事件处理函数​

  • @click="handleClick(msg)"

  • @click="handleClick(msg, $event)": $event可以访问并传递原生的 js dom 事件

  • @click="handleClick1(), handlerClick2()":可以传递多个事件处理函数, 用,隔开

const app = Vue.createApp({
  data() {
    return {
      todos: [
        {
          id: 1,
          content: "待办1",
        },
        {
          id: 2,
          content: "待办2",
        },
        {
          id: 3,
          content: "待办3",

      ],

        v-for="todo in todos"

8. 常见处理键盘,鼠标事件的修饰符​

  • 键盘修饰符: .enter,.tab,.delete,.esc,.up,.down,.left,.right

  • 常见组合键修饰符: .ctrl,.alt,.shift,.meta(Win 是徽标键,Mac 是 Command 键)

  • 鼠标修饰符: .left,.middle,.right

  • .exact: 允许精确地控制修饰符组合触发的事件

const app = Vue.createApp({
  data() {
    return {

app.mount("#app");

  <div>
    <ul @mousedown.right="comments.reverse()">

  </div>
</div>

11. Vue 生命周期函数​

  • beforeCreate: 调用app.mount()创建 Vue 实例之后, createApp()中配置项生效之前调用
  • created: 在beforeCreate()之后,createApp()中配置项生效之前调用,这个时候 computed,method 和 watch 已经配置好了
  • beforeMount: 在 created 生命周期执行完成之后,应用还没有挂载到app.mount()指定的 html 元素上,在挂载之前会调用beforeMount()生命周期钩子,这个时候应用还没在页面上渲染出来
  • mounted: mounted()会在应用挂载到app.mount()指定的 html 元素后执行,这个时候应用已经在页面上渲染出来了
  • beforeUpdate: 当应用中 html 模板需要重新渲染时,例如 data 中的属性发生了变化,那么在刷新之前会执行beforeUpdate()生命周期钩子
  • updated: updated()是在数据更新之后,html 重新渲染完成之后调用
  • beforeUnmount: beforeUnmount()会在应用卸载前执行,调用app.unmount()会卸载组件,那么正式卸载前,先执行beforeUnmount()生命周期钩子,这个时候 Vue 还是正常的,可以在里面做一些清理收尾操作
  • unmounted: unmounted()生命周期会在应用正式卸载之后调用,这个时候跟应用有关的事件监听,或者指令绑定都已经卸载了
  • 常用的生命周期钩子
createdmounted
1.最早可以访问 this
2.可以修改 data 属性
3.请求远程数据		DOM 节点已挂载,可以修改和访问 DOM
const app = Vue.createApp({

8. watch 监听数据变化​

  • 使用 watch 定义监听器: watcher 可以是定时器,请求 API 等耗时操作
  • 监听器名字要和监听的属性名保持一致
  • 监听器接收属性旧值和新值作为参数
const app = Vue.createApp({
  data() {
    return {
      showAnswer: false,
      countDown: 5,
      timer: null,
    };
  },
  computed: {
    label() {
      return this.showAnswer ? "隐藏答案 " + this.countDown : "显示答案";
    },
  },
  methods: {
    toggleAnswer() {
      this.showAnswer = !this.showAnswer;
    },
  },
  watch: {
    showAnswer(newVal, oldVal) {
      if (newVal) {
        this.countDown = 5;
        if (this.timer) {
          clearInterval(this.timer);
          this.timer = null;
        }
        // 5秒后关闭答案
        this.timer = setInterval(() => {
          this.countDown -= 1;
          if (this.countDown === 0) {
            this.showAnswer = false;
            clearInterval(this.timer);
            this.timer = null;
          }
        }, 1000);
      }
    },
  },
});
app.mount("#app");
<div id="app">
  <div class="question">
    <p>问：Vue 是一个什么样的框架？</p>
    <p v-show="showAnswer" class="answer">
      答：Vue 是一套用于构建用户界面的渐进式框架。
    </p>
    <button @click="toggleAnswer">{{ label }}</button>
  </div>
</div>

9. computed, methods 和 watch 之间的区别​

  • computed 和 methods 之间的区别
    • 计算属性适合做一些简单的操作,结果会缓存
    • 计算属性可以直接在 html 中,像 data 中的属性一样使用
    • 方法适合做复杂的操作,通常作为事件监听或者公共业务逻辑
    • 方法可以当作普通 js 函数使用
  • computed 和 watch 之间的区别
computedwatch
简单的业务逻辑计算耗时操作和远程 API 加载
可以直接在 html 模板中使用不能在 html 模板中使用
响应 data 数据变化响应数据变化
有返回值/getter没有返回值
也可以使用 setters 修改 data 中的数据可以修改 data 中的数据
  • methods 和 watch 之间的区别
methodswatch
可以在 watch 中被调用不能调用
可以直接在 html 模板中使用不能在 html 模板中使用
响应 data 数据变化响应 data 数据变化
可以有返回值,也可以没有没有返回值
  • 应用场景
    • computed 适合根据 data 中的属性做一些简单的计算并返回结果,例如数组的排序筛选,只有 data 中的属性发生变化时,才会重新计算,其他情况会缓存结果
    • watch 适合在 js 中监听 data 的变化,并根据变化做一些耗时的操作
    • methods 适合更复杂的计算,通常作为事件处理函数使用或者是抽取公共的业务逻辑,当作普通的 js 函数

14. 在 inline-style 内联样式中绑定 data 数据​

  • 直接使用 v-bind 绑定
  • 也可以绑定计算属性,方法等
const app = Vue.createApp({
  data() {
    return {
      fontSize: 16,
    };
  },
  computed: {
    pStyle() {
      return {
        color: "white",
        fontSize: this.fontSize + "px",
      };
    },
  },
});
app.mount("#app");
<div id="app">
  <div>
    <p :style="pStyle">这是一个段落</p>
    <button class="btn" @click="fontSize += 1">+</button>
    <button class="btn" @click="fontSize -= 1">-</button>
  </div>
</div>

15. 根据条件添加或删除 class 样式​

  • 直接使用 v-bind 绑定
  • 可以绑定单个 class
  • 可以用数组绑定多个 class
  • 可以用对象根据条件添加/移除 class
const app = Vue.createApp({
  data() {
    return {
      textClass: "text-blue",
      stateClass: "danger",
      isHidden: false,
    };
  },
});
app.mount("#app");
<div id="app">
  <p :class="[textClass, stateClass, { hide: isHidden }]">这是一个段落</p>
  <button @click="isHidden = !isHidden">切换状态</button>
</div>
